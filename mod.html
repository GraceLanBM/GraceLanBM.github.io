How I Started

This project started because I saw there were plenty of character mods in the Steam Workshop for Idol Manager, but none of them included Project Sekai characters. Especially at that time, these characters had some new illustrations, and I wanted to include both the old and new versions!

How I Built the Mod

I began by searching and found the Idol Manager editor guide from their official post on itch. After reading through it, digging into the game's asset structure, and checking out a few other character mods as examples, I started to write JSON files for this mod.

Of course, it didn't all go smoothly. The most time consuming bug I ran into was that the mini portraits didn't match the character's head. After some tests, I figured out it was because the game automatically crops the avatar from the illustration. So I had to manually set coordinates for each character in the JSON file to make the game crop the correct part. Then came another problem that the same coordinates didn't apply to different illustrations of the same character. To fix this bug, I decided to modify the illustrations to make their heads appear around the same spot. Finally, after a long time of work, the game showed the correct avatar for all characters and their different illustrations!

Language Support

I also wanted the mod to be accessible to fans in multiple regions, so I added support for three languages, included English, Chinese and Japanese. It was a bit of extra work, but totally worth it! Over 1000 players across different regions ended up subscribing to this mod!

Steam Workshop Link: English version, Chinese version, Japanese version

Future Plans (Maybe)

I haven't touched the mod in a while, and I don't think I'll have time to update it anytime soon — but I've still got a few ideas about it! Maybe give the characters some custom events, etc...

项目的开始

这个项目起源于我玩《偶像经理人》，在几十个小时后，我开始在创意工坊看看有什么角色模组以给游戏带来点新鲜感！在看到一众偶像类音乐游戏的角色时，我想起了世界计划这款手机音游。我本身就很喜欢miku，歌单里也有不少Vocaloid的歌，自然也在世界计划上线时就玩了。总之，在创意工坊我没找到能用的模组，在加上当时这些角色又有了新立绘！我就想，那干嘛不自己做一个呢？

制作模组

说干就干，我先在网上搜索了一下，在itch.io上找到了《偶像经理人》官方给出的Idol Manager editor guide（Google文档）！里面详细的解释了角色文件的位置和JSON文件里各参数变量等信息！跟着文档，我找到了相应位置，又下载了一些其他角色模组的文件来参考。在了解游戏的各种资产文件后，我又试着修改了各种文件和里面的参数，然后去游戏中看相应的改变。来回玩了几回之后，我觉得了解的差不多了，就开始正式写我想做的角色模组了！

当然，自己写个模组出来这种事肯定不可能一帆风顺。其中让我花了最长时间的bug不是卡代码，而是需要用Photoshop改图！在写完JSON文件并将角色立绘图放进相应文件夹位置后，我打开游戏测试，就发现了这个bug。角色图显示很正常，在正中间，但在半身图和头像图上却有偏左或偏右等问题。我检查了角色文件里写的参数，又调整测试了一下，发现半身图和头像图都是游戏运行时从立绘图片中自动裁剪的。那么要保证这两种图片裁剪到正确的位置，就需要调整设置裁剪的参数适用于不同的图片！但因为每个立绘的头部位置都不太一样，而同一个角色只能设置一个头部位置参数，为了让同一个角色在切换不同立绘后的头像处在正确位置，我只好用Photoshop自己调整角色图片，尽量让两个立绘的头部在同一位置。最后，在细致的调整后，游戏裁剪的半身图和头像都正确的切到了正确的位置了！

多种语言支持

因为世界计划这款游戏本身就有日服、美服以及台服（现在也出了国服！），所以我又做了英文和日文的版本！虽然多了点工作量，但发布后看到各个地区加起来有1000多名玩家的订阅，想到大家也许是和我一样的喜欢这些角色的粉丝，就觉得很开心QwQ！
