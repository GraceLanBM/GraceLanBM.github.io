How I Started

This project started because I saw there were plenty of character mods in the Steam Workshop for Idol Manager, but none of them included Project Sekai characters. Especially at that time, these characters had some new illustrations, and I wanted to include both the old and new versions!

How I Built the Mod

I began by searching and found the Idol Manager editor guide from their official post on itch. After reading through it, digging into the game's asset structure, and checking out a few other character mods as examples, I started to wrote JSON files for this mod.

Of course, it didn't all go smoothly. The most time consuming bug I ran into was that the mini portraits didn't match the character's head. After some tests, I figured out it was because the game automatically crops the avatar from the illustration. So I had to manually set coordinates for each character in the JSON file to make the game crops the correct part. Then came another problem that the same coordinates didn't apply to different illustrations of the same character. To fix this bug, I decided to modify the illustrations to make their heads appear around the same spot. Finally, after a long time of work, the game showed the correct avatar for all characters and their different illustrations!

Language Support

I also wanted the mod to be accessible to fans in multiple regions, so I added support for three languages, included English, Chinese and Japanese. It was a bit of extra work, but totally worth it! Over 670 players across different regions ended up subscribing to this mod!
