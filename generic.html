How I Start

This project started because I saw there are plenty of character mods in steam workshop, but none of them included Project Sekai characters especially at that time these characters have new illustrations and I wanted both old and new illustrations for them.

How I Built the Mod

I began by searching and found Idol Manager editor guide. After reading the guide, digging into the game's asset structure, and checking other character mods' as examples. I extrat and format the characters' illustrations in the correct folder and wrote the json files for them.

Of course, things didn't always go smoothly. The first and the most time consuming bug I met was the small pictures didn't match the avator. After some tests, I found that because the game automatically extract the character's illustration for the avator. So I need to add specific coordinates for each character in json file to make the game cut the correct place. Then the problem became the coordinates didn't reflect to different illustrations of one character. To fix this bug, I decided to modify the illustration to make their avators around the same coordinates. Finally, after this change, the game shows the correct avator for all characters!

Further more

I also wanted the mod to be accessible to fans in muultiple regions, so I added support for three languages, included English, Chinese and Japanese. It was a bit of extra work, but totally worth it! Over 670 people ended up subscribing to the mod!
